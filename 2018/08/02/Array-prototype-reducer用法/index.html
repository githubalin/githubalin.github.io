Array.prototype.reducer用法

2018.08.02 Alin JavaScript  热度 ℃

Array.prototype.reduce

第一个参数是累计的值，第二个参数是当前的值，第三个是下标，第四个是数组本身
reduce还可以传入一个初始值
用途：

1、数组求和

2、将二维数组转成一位数组

3、计算数组中元素出现的个数

4、数组去重

btw：数组去重可以用Set
